# Settler CLI

Command-line interface for Settler.

## Installation

```bash
npm install -g @settler/cli
```

Or using Homebrew:

```bash
brew install settler-cli
```

## Usage

```bash
# Reconcile transactions from a file
settler reconcile --file transactions.csv

# Sync data between sources
settler sync --source db --target api

# View help
settler --help
```

See the [full documentation](https://docs.settler.dev/cli) for more details.
